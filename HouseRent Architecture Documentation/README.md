# HouseRent - Architecture Documentation

This folder contains the architecture and design documents prepared as part of the **SkillWallet Full Stack (MERN) Internship Project**.

## Documents Included

1. ER Diagram
   - Database entity relationships used in the HouseRent application.

2. Features
   - Functional features implemented in the project.

3. MVC Architecture
   - Model-View-Controller architecture followed in the application.

4. Roles and Responsibilities
   - Description of Admin, Owner, and Renter modules and their responsibilities.

5. Technical Architecture
   - Overall system architecture, frontend, backend, database, and API flow.

6. User Flow
   - User journey and navigation flow across the application.

These documents were created during the SkillWallet project planning and architecture phase and describe the design and implementation of the HouseRent application.
